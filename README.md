# Lista de Tarefas com JavaScript

Aplicacao de lista de tarefas feita com **HTML, CSS e JavaScript puro**, com foco em manipulacao do DOM, filtros, busca e persistencia local.

## Visao Geral

Este projeto foi desenvolvido como pratica de frontend para criar uma To-Do List funcional sem frameworks, trabalhando com eventos, renderizacao dinamica e armazenamento no navegador.

## Funcionalidades

- Adicionar novas tarefas
- Editar tarefas existentes
- Remover tarefas da lista
- Marcar tarefas como concluidas
- Buscar tarefas em tempo real
- Filtrar por `todas`, `ativas` e `concluidas`
- Limpar tarefas concluidas
- Salvar tarefas com `localStorage`

## Tecnologias

- HTML5
- CSS3
- JavaScript
- DOM API
- `localStorage`

## Como Executar

1. Clone este repositorio:

```bash
git clone https://github.com/victorbernardo18-oss/PROJETO-LISTA-DE-TAREFAS-COM-JS.git
```

2. Entre na pasta do projeto:

```bash
cd PROJETO-LISTA-DE-TAREFAS-COM-JS
```

3. Abra o arquivo `index.html` no navegador.

Se preferir, rode um servidor local simples:

```bash
python -m http.server 5500
```

Depois, abra `http://localhost:5500`.
